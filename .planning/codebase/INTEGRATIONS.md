# Integrations - Sistema La Casa de los Niños

## Local Storage
- **SQLite**: The application uses a local SQLite database file (`casaninos.db`) for all data persistence. 
- **Assets**: Embedded and local assets (fonts, images) are managed via the project's asset directory.

## External Resources
- **Google Fonts**: Grandstander and Nunito Sans are integrated locally into the application assets.
- **FontAwesome**: Integrated via the Sharp library for vector-based, high-fidelity icons in WinForms.

## Third-Party APIs
- No external cloud APIs or services are currently integrated. The system is designed for offline local operation.
