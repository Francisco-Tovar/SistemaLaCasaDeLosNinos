# Technology Stack - Sistema La Casa de los Niños

## Core Framework
- **Runtime**: .NET 8.0 Windows
- **UI Framework**: Windows Forms (WinForms)

## Data Access & Storage
- **Database**: SQLite (local)
- **Data Provider**: Microsoft.Data.Sqlite
- **ORM**: Dapper (v2.1.72)

## UI/UX Libraries
- **Icons**: FontAwesome.Sharp (v6.6.0)
- **Typography**: 
  - **Grandstander**: Used for titles and headers.
  - **Nunito Sans**: Used for body text and general interface.

## Infrastructure & Configuration
- **Dependency Injection**: Microsoft.Extensions.DependencyInjection
- **Configuration**: Microsoft.Extensions.Configuration (Json support for appsettings)
- **Logging**: Simple logging to local folder (logs/).
